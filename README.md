# Multiple Data Compression and Encryption

## Project Overview
This project implements multiple data compression and encryption techniques using iterative methods. It demonstrates how different compression algorithms can reduce file size and how encryption ensures secure data storage.

## Technologies Used
- C Programming
- File Handling
- Data Compression Techniques
- Encryption & Decryption

## Features
- File Compression
- File Decompression
- Encryption
- Decryption
- Efficient storage using compression algorithms

## Project Structure
compress2.c – Compression algorithm module  
compress3.c – Compression technique implementation  
compression.c – Main compression logic  
deCompression.c – Decompression module  
dataStructures.h – Data structure definitions  
Makefile – Project compilation file

## How to Run
1. Compile the program using: make
2. Run the program using: ./compression
